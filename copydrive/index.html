Project Overview

The GoogleDrive Cloner uses the Google Drive V3 API to perform "Cloud-to-Cloud" transfers. Unlike traditional methods that download files to your device before re-uploading them, this tool sends a "Copy" instruction directly to Google’s data centers.

Zero Data Consumption

Operations occur on Google's infrastructure. Local internet is only used for JSON commands.

Automated Recursion

Deep-scans folders and recreates the entire directory structure in the target location.

Troubleshooting Base

Error 403: Forbidden (Insufficient Permissions)

Cause: The logged-in account lacks "Editor" access to the Target folder.
Fix: Ensure you own the destination or have Editor rights.

Error 404: File Not Found

Cause: Source ID is incorrect or folder is not shared.
Fix: Ensure source is set to "Anyone with the link can view".
